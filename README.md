# IRIS

Plateforme de gestion des avis clients pour Pierre & Vacances Center Parcs (PVCP).

## Stack

React 18 (Vite + TypeScript) · Tailwind CSS · Zustand · React Router v6 · Recharts · Sonner · lucide-react

## Démarrage

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — serveur de développement
- `npm run build` — build de production
- `npm run lint` — lint (oxlint)
- `npm run preview` — prévisualiser le build
